# Methodology

## 1. Overview
This project develops a data-driven municipal financial risk scoring model using publicly available financial and demographic data. The objective is to identify early warning signals of fiscal stress, debt burden, and financial transparency risk in U.S. municipalities.

---

## 2. Scope of Analysis and Limitations

### Geographic Scope
Illinois municipalities

### Sample Size
50–100 municipalities

### Time Period
FY2022–FY2024

### Limitations
- Based on public data only
- Not an audit opinion
- Screening tool only

## 3. Intended Use

This model is designed as an early-warning analytical tool to support auditors, policymakers, and public stakeholders in identifying municipalities that may exhibit elevated financial risk indicators.

It is not intended to replace professional judgment, audit procedures, or formal financial assessments.

## 4. Future Enhancements

- Expand to additional states
- Incorporate multi-year trend analysis
- Add pension and long-term liability indicators
